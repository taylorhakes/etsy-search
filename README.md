etsy-search
===========

Search the Etsy API with a simple webpage

[![Build Status](https://travis-ci.org/tayorhakes/etsy-search.png)](https://travis-ci.org/tayorhakes/etsy-search)

Documentation
-------------
Dependencies
```
Node
NPM
Bower
```

To install run
```
npm install
bower install
```

To run tests
```
grunt test
```

To start node server for viewing app
```
grunt server
```
